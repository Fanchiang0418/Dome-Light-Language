# Dome-Light-Language


此子計畫接續舞蹈動作語言化後的文字結果，將具語意的文字符號與敘事描述進一步轉換為燈光表現。系統會依據文字中的關鍵詞，對應至 LED 燈光的亮度、顏色、方向、速度、閃爍與漸層等參數，使燈光成為承接舞蹈語意的第二層媒介。
